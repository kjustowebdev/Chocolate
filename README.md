# Chocolate EDA

<br>**Overview**</br>

The dataset used in this project is sourced from Kaggle (Chocolate Sales Dataset). The objective of this analysis is to demonstrate exploratory data analysis (EDA) by examining sales data from a business perspective. The project focuses on the use of aggregations and ranking techniques to quickly identify patterns in product consumption and spending behavior. Interactive visualizations are created using Plotly to present insights in a clear and visually engaging manner.


[Dataset Link](https://www.kaggle.com/datasets/atharvasoundankar/chocolate-sales)
